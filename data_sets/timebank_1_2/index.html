Timebank 1.2


Introduction

This file contains documentation on version 1.2 of the Timebank corpus, Linguistic Data Consortium (LDC) catalog number LDC2006T08 and isbn 1-58563-379-8.

The TimeBank Corpus contains 183 news articles that have been annotated with temporal information, adding events, times and temporal links between events and times. The annotation follows the TimeML 1.2.1 specification.
